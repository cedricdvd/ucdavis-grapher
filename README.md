# UC Davis Grapher

## Basic Setup

Set up Virtual Environment

```zsh
source .venv/bin/activate
```

Scrape Data

```zsh
python manage.py scrape
```


Run Backend Server

```zsh
python manage.py runserver
```

Run Frontend

```zsh
npm start
```
